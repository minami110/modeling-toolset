# modeling-toolset

## Install

- Open `modeling-toolset.json` and edit here

```md
"MODELING_TOOLSET": "<ENTER_YOUR_INSTALL_PATH>/modeling-toolset"
```

- Then copy `modeling-toolset.json` into `$HOUDINI_USER_PREF_DIR/packages/`
